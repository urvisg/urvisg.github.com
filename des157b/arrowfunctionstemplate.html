Fun with Addition

This function accepts two numbers as parameters and returns the sum of the numbers.

Named Function

            function addition(num1, num2){
                return num1 + num2;
            }

Function Expression

            const addition = function(num1, num2){
                return num1 + num2;
            }

Arrow Function

            const addition = (num1, num2) => num1 + num2;
